# 开始使用

为了开始使用 SubConv，您可以将其部署在 Vercel 上、通过 Docker 部署，或者部署在您自己的 VPS 上。如果您想先试用一下，可以使用仓库 README 中提供的 demo。

## 部署
- [在 Vercel 上部署](/zh_CN/guide/deploy/vercel)  
- [在 Docker 上部署](/zh_CN/guide/deploy/docker)
- [在 VPS 上部署](/zh_CN/guide/deploy/vps)  

## 使用
打开您部署好的 SubConv 链接后，您会看到 Web UI。它会帮助您生成新的 mihomo 订阅链接。
